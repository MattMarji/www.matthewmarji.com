Desktop Redesign!

The Desktop Client has come a long way. I have created an entire new UI layout. We have introduced graphs for paid and free plans. On free plans, users' daily bandwidth is tracked. On paid plans, the number of ad trackers we block is graphed. Also we introduced a new icon set and in-app notifications (such as an Update is Available!)

Functionality of the VPN continues to improve. We are set to run functionally on OS/X 10 and Windows 10 in the near-future. It has been an amazing experience redesigning the client and hearing user feedback. Leave me some feedback at: matthew@surfeasy.com

My journey thus far.

The Desktop Client uses the Qt Framework. Developing user-facing UI can be difficult. Minor bugs make something that may be functional look completely broken. I have learned about the OpenVPN protocols, and third-party C++ libraries such as boost.

I have been given the ability to take full control of how the desktop looks and how it functions. Improving the user experience on Mac and Windows was a large priority once I began work at SurfEasy. Over the first 5 months, I developed an improved UI client that is now available for download!
